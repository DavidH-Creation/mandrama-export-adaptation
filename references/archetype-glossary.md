# Character Archetype Glossary

Reference lookup table for translating Chinese drama character archetypes into globally legible functional roles. Use during Layer 2 (character archetype translation).

Choose the mapping that fits the story's tone — prestige drama, romance, comedy, and thriller each need different translations of the same archetype.

---

|---|---|---|
| **霸总** (bà zǒng) | Domineering CEO | Cold founder · Ruthless heir · Emotionally armored billionaire · Obsessive corporate predator |
| **白月光** (bái yuè guāng) | White moonlight | Idealized first love · Untouchable past obsession · The one that got away |
| **绿茶** (lǜ chá) | Green tea | Manipulative ingenue · Performative innocent · Two-faced rival · Sweet-faced schemer |
| **赘婿** (zhuì xù) | Live-in son-in-law | Dependent husband · Outsider husband · Strategic marriage underdog · Underestimated spouse |

## Usage Notes

- **Always choose by tone.** "霸总" in a prestige thriller is "ruthless heir." In a romance, it's "emotionally armored billionaire." In a comedy, it might be "control-freak CEO."
- **Never use the Chinese label as-is** in export material unless the target audience is sinophone or the project deliberately preserves Chinese cultural texture.
- **Compound archetypes are common.** A character might be 霸总 + 腹黑 + 黑化 — map the combination, not each label separately.
- **Extend this glossary** when encountering new archetypes in source material. Add entries using the same format.
