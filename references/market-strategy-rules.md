# Market Strategy Rules

Use this reference in Phase 2 (route selection) and Phase 5 (format and packaging decisions).

---

## Market Routes

### Route A — English-Led Premium Export (Default)

**Use when:**
- Premise is universal and easy to pitch in one sentence
- Characters can be legible to Western audiences after adaptation
- Sponsorability matters
- Project benefits from realism or prestige packaging

**Default package:**
- English dub / native-sounding English dialogue
- English thumbnails, titles, metadata
- English-first character naming or globally legible naming
- Globally understandable setting cues

### Route B — Japanese-Performance + Premium English Subtitles

**Use when:**
- Project is anime-coded, otaku-coded, fantasy-coded, idol-coded, or stylized melodrama
- Subculture authenticity matters more than realism
- Performance tone benefits from Japanese cadence

**Default package:**
- Japanese voice performance
- Premium English subtitles (not machine-style captions)
- Metadata optimized for global anime audience in English

### Route C — Multilingual Layered Release

**Use when:**
- Asset reuse is high
- Dubbing cost is justified by market size
- Testing multiple geographies matters

**Recommended sequence:**
1. English master
2. Japanese secondary OR Spanish secondary
3. Localized subtitles beyond that

---

## Route Selection Criteria

Evaluate each route against:

| Criterion | Route A | Route B | Route C |
|---|---|---|---|
| **Audience reach** | Broadest English-speaking market | Anime/subculture niche (large but specific) | Multi-market but diluted focus |
| **Monetization** | Highest CPM ad markets | Strong merch/fan economy | Volume-dependent |
| **Dubbing burden** | High (needs native-quality English) | Medium (Japanese perf + subtitle craft) | Very high (multiple dubs) |
| **Sponsorability** | Highest | Moderate (brand depends on niche) | Moderate |
| **Authenticity risk** | Low if well-adapted | Low if genre fits | Medium (neither fish nor fowl) |
| **Cost** | Medium | Low-medium | High |

Choose ONE route. Justify against all criteria. Document rejected routes and why.

## Production Format Rules

### 3D Photoreal / MetaHuman Path

**Recommend only if ALL or MOST are true:**
- Story tone benefits from realism, tension, status, luxury, or thriller aesthetics
- Audience is expected to reward cinematic presentation
- Consistent character performance is important
- Production team can manage UE lighting, rigging, animation, facial capture, and shot continuity
- The project wants a premium visual moat

**Do NOT recommend by default for:**
- High-camp melodrama that benefits from exaggeration
- Heavily stylized fantasy better served by anime language
- Low-budget teams without shot pipeline discipline
- Dialogue-heavy episodes where uncanny-valley risk would kill immersion

### 2D / Anime / Motion-Comic Path

**Recommend when:**
- Emotional exaggeration helps the material
- Turnaround speed matters more than realism
- Audience is subculture-native
- Stylization hides production weakness better than failed realism

### Hybrid Path

**Recommend when:**
- Project has mixed tonal demands (e.g., realistic drama with fantasy sequences)
- Budget allows selective 3D for key scenes with 2D for bulk episodes
- Proof-of-concept phase where testing both approaches matters

---

## Format Decision Checklist

Before finalizing format recommendation:

- [ ] Does the visual format match the story's emotional tone?
- [ ] Can the production team execute this format at the required quality?
- [ ] Does the format choice align with the target audience's expectations?
- [ ] Is there uncanny-valley risk that would hurt immersion?
- [ ] Does the budget support the chosen format for the full episode run?
